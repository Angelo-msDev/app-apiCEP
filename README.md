# app-apiCEP

📍 App Consultar CEP
Uma aplicação web moderna e responsiva para consulta de endereços através do CEP, consumindo a API pública ViaCEP. O projeto foi desenvolvido para oferecer uma experiência de usuário fluida, com validações em tempo real e interface limpa.

🚀 Demonstração
Acesse o projeto online: https://app-api-cep.vercel.app/

✨ Funcionalidades
Consulta Instantânea: Busca automática de endereço ao preencher o CEP.

Validação de Dados: Impede a entrada de caracteres inválidos e verifica o formato do CEP (8 dígitos).

Feedback Visual: Indica quando um CEP não foi encontrado ou se há erro de conexão.

Design Responsivo: Totalmente adaptado para dispositivos móveis, tablets e desktops.

Limpeza de Formulário: Botão dedicado para limpar os campos e realizar uma nova busca.

🛠️ Tecnologias Utilizadas
O projeto foi construído utilizando:

HTML5: Estrutura semântica da página.

CSS3: Estilização com foco em Flexbox e Grid para responsividade.

JavaScript (ES6+): Lógica de consumo de API (Fetch API) e manipulação do DOM.

ViaCEP API: Webservice gratuito para consulta de CEP do Brasil.

Basta abrir o arquivo index.html em seu navegador de preferência ou utilizar a extensão Live Server do VS Code.

📖 Como usar
Digite o número do CEP no campo indicado (apenas números).

O sistema fará a busca automaticamente (ou após sair do campo/clicar em buscar).

Os campos de Logradouro, Bairro, Cidade e Estado serão preenchidos automaticamente.

Caso queira pesquisar outro endereço, utilize o botão de limpar.
